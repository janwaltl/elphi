◆ get_next()

std::optional< Sample > elphi::data::PerfCpuSampler::get_next ( )

Retrieve the next sample, non-blocking.

See also
get_poll_entry() to poll until samples are generated.
Warning
Samples are gathered by perf_event which uses ring buffer, therefore there is hard limit on number of pending samples unextracted samples. One must call get_next() often enough to not lose any.
Returns
Sample
Return values
Nothingif not called within start(), end() or there is no sample.
